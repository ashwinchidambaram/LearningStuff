Q3: Which of the following is not a recommended strategy to take care of missing data ?

  Answer: Delete the observation that contains the missing data.

  Recommended Methods:
    - Replace missing data with the median of the feature column
    - Replace missing data with the mean of the feature column
    - Replace missing data with the most frequent value of the feature column
